---
title: "Platform Authentication"
description: "Sign up and log in as a CoreBase developer."
---

<Note>
  This is for *you*, the developer building on CoreBase — not your app's end users.
  For your app's own sign-up and sign-in flows, see [User Accounts](/authentication).
</Note>

Most people create their account once through the CoreBase Dashboard and never touch this
programmatically. This API exists mainly for CLI tools and automation.

## Sign Up

```typescript
const { data, error } = await corebase.platformAuth.signUp({
  email: 'you@example.com',
  password: 'securePassword123',
  name: 'Jane Developer',
});
```

## Log In

```typescript
const { data, error } = await corebase.platformAuth.signIn({
  email: 'you@example.com',
  password: 'securePassword123',
});
```

## Get the Current Developer

```typescript
const { data, error } = await corebase.platformAuth.getUser();
```

## Refresh the Session

```typescript
const { data, error } = await corebase.platformAuth.refreshSession();
```
